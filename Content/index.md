# 知识库

> [!abstract] 概览
> 本知识库基于 Karpathy 知识编译模式构建，采用 Obsidian + Quartz 架构，是一个持续生长的"活体维基"。所有笔记遵循统一规范，通过 Ingest → Query → Lint → Index 流水线实现知识复利。

## 知识编译流水线

| 操作 | 目的 | 频率 |
|:-----|:-----|:-----|
| **Ingest** | 读取新笔记，提取概念，更新 Wiki | 每学完一个章节 |
| **Query** | 基于已编译 Wiki 回答问题 | 日常学习 |
| **Lint** | 扫描矛盾、孤立、缺失 | 每周一次 |
| **Index** | 维护 Wiki Index 和日志 | 每次操作后 |
