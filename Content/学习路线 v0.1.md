# 学习路线 v0.1

> [!abstract] 概览
> 本文档记录基于仓库中 6 本教材制定的三阶段学习路线。核心策略：先逻辑学打基础 → 离散数学与数据结构黄金同步 → 操作系统与 CSAPP 系统深化。

## 三阶段路线总览

```
阶段一：数学基础期
  离散数学 第1-5章（逻辑→集合→算法→数论→归纳）
  + 逻辑学导论 第1-4章（基本概念→论证分析→语言定义→谬误）

阶段二：黄金同步期
  离散数学 第6-12章 ←→ 数据结构 第1-12章
  图论↔图、树↔二叉树完美互补

阶段三：系统深化期
  现代操作系统 全书
  + CSAPP 全书
```
