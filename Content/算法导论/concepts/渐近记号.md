# 渐近记号
> [!abstract] 概述
> ==渐近记号== 是描述函数增长率之间关系的数学工具统称，包括 $O$、$\Omega$、$\Theta$、$o$、$\omega$ 五种记号。

## 定义
> [!def] 五种渐近记号
> - **$O$ 记号**：渐近上界 — [[算法导论/concepts/大O记号]]
> - **$\Omega$ 记号**：渐近下界 — [[算法导论/concepts/大Omega记号]]
> - **$\Theta$ 记号**：渐近紧界 — [[算法导论/concepts/大Theta记号]]
> - **$o$ 记号**：严格上界（非渐近紧）
> - **$\omega$ 记号**：严格下界（非渐近紧）

## 核心性质
- $f(n) = \Theta(g(n))$ 当且仅当 $f(n) = O(g(n))$ 且 $f(n) = \Omega(g(n))$。
- $o$ 与 $\omega$ 类似于 $O$ 与 $\Omega$，但不允许等号（严格不等式），极限形式：$\lim_{n \to \infty} f(n)/g(n) = 0$（$o$）或 $\infty$（$\omega$）。
- 渐近记号可嵌套使用，如 $O(n \lg n + n^2) = O(n^2)$。
- 传递性：若 $f = O(g)$ 且 $g = O(h)$，则 $f = O(h)$。
