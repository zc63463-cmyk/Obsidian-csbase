# 直言三段论有效性判定定理

> [!abstract] 概述
> ==直言三段论有效性判定定理==（Syllogism Validity Theorem）：一个直言三段论是有效的，当且仅当在文恩图检验中，当两个前提都为真时结论所断言的区域确实被阴影覆盖或标记为非空。等价地，它也可以通过三段论的五条规则来判定。

## 定理陈述

> [!def] 形式化陈述
> **定理**（直言三段论有效性判定定理）：设一个标准形式的直言三段论由大前提、小前提和结论组成，分别涉及大项 $P$、小项 $S$ 和中项 $M$。则该三段论有效的充分必要条件是以下两种判定方法之一通过：
>
> **方法一（文恩图法）**：画出三个相交的圆（代表 $S$、$P$、$M$），依次将两个前提所断言的信息标示到图上（全称命题画阴影，特称命题标×），然后检查结论所断言的内容是否已被图示所蕴含。
>
> **方法二（规则法）**：三段论同时满足以下五条规则：
> 1. 中项至少在一个前提中周延
> 2. 在结论中周延的项必须在前提中也周延
> 3. 至少有一个前提是肯定的
> 4. 如果结论是肯定的，则两个前提都必须是肯定的
> 5. 如果结论是特称的，则至少有一个前提是特称的

**各项说明：**
- **周延**（distribution）：一个词项在某命题中是周延的，当且仅当该命题对该词项所指类的全部元素做出了断言
- **中项**（middle term）：在两个前提中都出现但不在结论中出现的词项 $M$
- **大项**（major term）：结论的谓项 $P$
- **小项**（minor term）：结论的主项 $S$

## 证明概要

> [!info] 证明思路（文恩图三圆交集判定）

### 核心思想

文恩图方法将三段论的有效性问题转化为几何判定问题：三个圆将平面分成8个区域，分别对应 $S$、$P$、$M$ 的各种组合。前提的信息被编码为阴影（空集）或×（非空），结论的有效性取决于其断言是否必然被前提的图示所蕴含。

### 详细步骤

**第一步：构造三圆文恩图**

三个相交的圆将平面分为8个区域：

| 区域 | 含义 |
|:---:|:---|
| $SPM$ | 同时属于 $S$、$P$、$M$ 的元素 |
| $SP\overline{M}$ | 属于 $S$ 和 $P$ 但不属于 $M$ 的元素 |
| $S\overline{P}M$ | 属于 $S$ 和 $M$ 但不属于 $P$ 的元素 |
| $\overline{S}PM$ | 属于 $P$ 和 $M$ 但不属于 $S$ 的元素 |
| $S\overline{P}\overline{M}$ | 仅属于 $S$ 的元素 |
| $\overline{S}P\overline{M}$ | 仅属于 $P$ 的元素 |
| $\overline{S}\overline{P}M$ | 仅属于 $M$ 的元素 |
| $\overline{S}\overline{P}\overline{M}$ | 三个类都不属于的元素 |

**第二步：标示前提信息**

- **全称肯定（A）**："所有X是Y"→ 将 $X\overline{Y}$ 区域画阴影
- **全称否定（E）**："没有X是Y"→ 将 $XY$ 区域画阴影
- **特称肯定（I）**："有X是Y"→ 在 $XY$ 区域标×
- **特称否定（O）**："有X不是Y"→ 在 $X\overline{Y}$ 区域标×

注意：标×时如果该区域已被分为两部分（被另一圆分割），应将×标在两个子区域的边界线上，表示至少有一个子区域非空。

**第三步：检验结论**

检查结论所断言的内容是否已被图示蕴含：
- 若结论是全称命题，检查相应区域是否已完全被阴影覆盖
- 若结论是特称命题，检查相应区域（或其子区域）是否已标有×

**第四步：等价性说明**

文恩图法与规则法是等价的判定方法。五条规则分别对应：
- 规则1防止中项不周延谬误
- 规则2防止大项不当周延和小项不当周延谬误
- 规则3和4防止从否定前提推出肯定结论
- 规则5防止从全称前提推出特称结论（在布尔解释下）

证毕。$\blacksquare$

## 关键推论

- **推论1（有效式的穷举）**：在4个格 x 64个式 = 256个可能的三段论形式中，仅有15个是有效的（亚里士多德解释下），在布尔解释下减少为15个中的部分有效式。详见 [[逻辑学/concepts/直言三段论的15个有效形式]]。
- **推论2（文恩图法与规则法的等价性）**：两种方法对任何标准形式的三段论给出相同的判定结果，它们是同一有效性概念的不同表征方式。
- **推论3（非标准三段论的处理）**：对于非标准形式的三段论，需要先通过翻译、换位、换质等操作将其化为标准形式，再应用本定理进行判定。

## 应用场景

1. **三段论有效性检验**：在 [[逻辑学/concepts/直言三段论|直言三段论]] 的学习中，本定理提供了两种系统化的有效性判定方法，是三段论理论的核心工具。
2. **逻辑谬误识别**：通过规则法可以精确识别三段论中的各种谬误，如"中项不周延"、"大项不当周延"等。
3. **日常论证分析**：将日常语言中的三段论论证标准化后，应用本定理判定其有效性，是批判性思维的重要技能。
4. **形式逻辑教学**：文恩图法因其直观性成为教学中最常用的三段论判定方法，规则法则因其简洁性适合快速判定。
